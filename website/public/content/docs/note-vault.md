# Note Vault

The **Note Vault** is the core of your digital brain in Synabit. It's where you store, organize, and connect all your thoughts, ideas, and long-form writing.

Synabit uses an ultra-fast, local-first SQLite database to store your notes, meaning you can search through thousands of notes instantly without any network lag.

---

## The Editor

Synabit features a distraction-free, block-based editor that supports full Markdown syntax along with rich media embeddings.

### Markdown Support
You can write entirely using standard Markdown, or use the `/` (slash) menu to quickly insert blocks.

- `# Heading 1`
- `**Bold**` and `*Italics*`
- `> Blockquotes`
- `[Links](url)`

### Bi-directional Linking
The true power of the Note Vault lies in bi-directional linking. 

By typing `[[` anywhere in the editor, you can link the current note to another note in your vault. 
If the note doesn't exist yet, Synabit will create it for you. This allows you to build a personal wiki or a Zettelkasten system organically.

---

## Organizing Notes

While folders are supported, Synabit encourages you to organize your notes using **Tags** and **Links**.

- **Tags:** Add `#tagname` anywhere in your note to instantly categorize it.
- **Graph View:** Synabit provides a visual Graph View that maps out how all your notes are connected, helping you discover hidden relationships between your ideas.

## Data Portability

Your notes are never locked in. You can export your entire vault as plain `.md` files at any time by going to **Settings > Export > Export as Markdown**.
