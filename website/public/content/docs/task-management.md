# Task Management

A digital brain isn't just for storing thoughts; it's also for getting things done. Synabit deeply integrates task management directly into your workflow so you never have to context-switch to another app.

---

## Tasks in Notes

The most frictionless way to create a task in Synabit is to type it directly inside a note.

By typing `- [ ]` or using the slash command `/task`, you can convert any block into an actionable item.
Synabit automatically scans your entire vault for these checkboxes and aggregates them into a centralized **Tasks** view.

### Task Properties
Tasks in Synabit are more than just checkboxes. By right-clicking a task or using keyboard shortcuts, you can assign:
- **Due Dates:** Assign a specific date or time.
- **Priority:** Mark tasks as High, Medium, or Low.
- **Context/Tags:** Tag a task to group it with a specific project.

---

## The Global Task View

While you can create tasks inside any note, the **Global Task View** (accessible via the left sidebar) provides a bird's-eye view of everything you need to do.

You can filter and sort this view by:
- What's due today.
- Tasks assigned to specific tags.
- Overdue tasks.

### Kanban Boards
If you prefer visual project management, any collection of tasks in the Global Task View can be transformed into a Kanban Board. You can drag and drop tasks between columns (e.g., *To Do*, *In Progress*, *Done*).

## Calendar Integration
Tasks with due dates automatically appear on your Synabit Calendar, ensuring that your planning and execution happen in the same unified environment.
